# Get DATRAS tables with latin names

Get DATRAS tables with latin names

## Usage

``` r
dr_get_data_latin(recordtype, survey, year, quarter, quiet = TRUE)
```

## Arguments

- recordtype:

  Any of HH, HL, CA or all

- survey:

  Name of the survey, e.g. "ROCKALL"

- year:

  A text string, like "1999:2009"

- quarter:

  A text string, like "1:4"

- quiet:

  A boolean (default TRUE) - suppress messages '

## Value

A tibble
