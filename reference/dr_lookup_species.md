# A table of english and latin species names and aphia

A data frame with over 2000 rows and 3 columns:

- aphia:

  Species id in the DATRAS exchange data

- latin:

  Species latin name

- species:

  Species english name

## Usage

``` r
dr_lookup_species
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
2019 rows and 3 columns.

## Source

R-package worrms and aphia in HL- and CA-tables
