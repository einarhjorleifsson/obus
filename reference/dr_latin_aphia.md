# Coastline covering majority of DATRAS data

A data frame with over 2000 rows and 3 columns:

- Valid_Aphia:

  Species id in the DATRAS exchange data

- latin:

  Species latin name

- species:

  Species english name

## Usage

``` r
dr_latin_aphia
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
2011 rows and 3 columns.
